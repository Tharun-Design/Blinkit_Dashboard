# 📊 Blinkit Sales Analysis Dashboard

## 🟢 Project Overview

This project presents an interactive **Power BI dashboard** analyzing sales performance of *Blinkit – India’s Last Minute App*.

The dashboard provides insights into:

- 📈 Sales performance
- 🏪 Outlet performance
- 🥫 Product category analysis
- ⭐ Customer rating trends
- 💰 Revenue distribution patterns

The goal of this project is to derive **business insights** that help in decision-making, revenue optimization, and operational improvement.

---

## 📂 Project File

- `Blinkit.pbix` → Power BI dashboard file

---

## 📊 Key Business KPIs

The dashboard includes the following major KPIs:

- **Total Sales**
- **Average Sales**
- **Number of Items Sold**
- **Average Rating**
- **Sales by Item Type**
- **Sales by Outlet Size**
- **Sales by Outlet Location**
- **Outlet Establishment Year Analysis**
- **Fat Content Analysis**

---

## 📌 Dashboard Features

✔ Interactive slicers  
✔ Dynamic filtering  
✔ Year-wise trend analysis  
✔ Location-based performance comparison  
✔ Category-wise revenue insights  
✔ Outlet type & size performance tracking

---

## 📈 Business Insights Generated

- Tier 3 outlets generated strong sales performance.
- Regular fat products contributed more revenue than low-fat.
- Medium-sized outlets performed better in revenue contribution.
- Certain item categories dominate overall revenue.

## 🚀 How to Use

1. Download the `Blinkit.pbix` file.
2. Open it using **Power BI Desktop**.
3. Interact with filters and slicers to explore insights.
4. Modify visuals or DAX measures if needed.
